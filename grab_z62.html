Język CSS dopuszcza stosowanie zarówno dużych jak i małych liter. Poprawnymi nazwami atrybutu są background, backGROUND oraz BACKground.

polski lang
CSS Kaskadowe Arkusze Stylów (z ang. Cascading Style Sheets) jest to specjalny język stworzony w celu estetycznego zarządzania sposobem formatowania (wyglądem) elementów znajdujących się na stronach WWW.

angielski lang
CSS - Cascading Style Sheets is a special language created to aesthetically manage the formatting (appearance) of elements on websites.
